Shortcuts:

cmd/win + alt + home = 'pause/play'
cmd/win + alt + end = 'stop'
cmd/win + alt + arrow up = 'volume up'
cmd/win + alt + arrow down = 'volume down'
cmd/win + alt + arrow right = 'next track'
cmd/win + alt + arrow left = 'previous track'

Pro Tip: 
You can replace cmd/win easy by ctrl by doing a 'replace all' 
on the python file and replacing 'cmd' for 'ctrl'.

Just be careful some windows computer turn the screen orientation when 'Ctrl'+'alt'+'arrow keys'
